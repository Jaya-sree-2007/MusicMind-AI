# 🎵 MusicMind AI

## 📌 Project Overview

MusicMind AI is a web-based music personality analysis system developed using Python, Streamlit, MySQL, and Machine Learning concepts. The application analyzes user responses through an interactive questionnaire and predicts personality traits based on music preferences and behavioral patterns. Based on the predicted personality, the system recommends suitable music genres and curated Spotify playlists to enhance the user's listening experience.

The platform also includes secure user authentication, personalized dashboards, analysis history tracking, PDF report generation, and administrative analytics for efficient system management.

## 🚀 Features

* 🔐 User Registration and Login System
* 🔑 Forgot Password & Password Reset Module
* 🛡️ Secure Password Encryption using Bcrypt
* 🎧 Music Personality Analysis
* 🎵 Genre-Based Music Recommendations
* 🔗 Spotify Playlist Recommendations
* 📊 Student Dashboard with Analysis History
* 📈 Score Trends and Personality Analytics
* 📄 PDF Report Generation
* 👨‍💼 Admin Dashboard

## 🎓 Project Objective
